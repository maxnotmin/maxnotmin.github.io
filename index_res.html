Web Development

I’m comfortable creating websites from the ground up or taking over existing sites and figuring out how they work. During my time at Nationwide Insurance I’ve created and maintained sites built with Grails and Python as well as static HTML sites. From programming database queries and server deployments to creating graphics and writing CSS, I’ve touched just about every part of the application life cycle.

adminme.nwie.net

This internal Nationwide is called Adminme. It allows engineers to elevate their privileges to certain machines. Using Python Flask and BackboneJS we were able to create and deploy a robust tool able to communicate with multiple domains. This site was required to be IE10 and Firefox compatible.

Technologies Used: Python, Flask, BackBoneJS, SASS, PostgreSQL

This site was deployed to a Windows server using IIS and FastCGI with the website using a PostgreSQL DB.

cobralabs.nwie.net

CobraLabs is an internal Nationwide site that displays basic information about the services we provided as well as a form to allow customers to request automation consultation. The frontend theme we designed and developed allowed us to quickly writeframe new projects and conformed to Nationwide brand standards.

Technologies Used: Python, Flask, BackBoneJS, SASS, SQLite

This site was deployed to a linux server using Apache with the website using a SQLite DB.

Automation Work

During my time working within Infrastructure and Operations the teams I were apart of focused automating redundant tasks and finding ways to ways to give time back to people. Typically using a combination Python, Javascript, REST, APIs and Orchestration software allowed complexed tasks to be streamlined.
