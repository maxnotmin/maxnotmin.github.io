Web Development

I’m comfortable creating websites from the ground up or taking over existing sites and figuring out how they work. During my time at Nationwide Insurance I’ve created and maintained sites built with Grails and Python as well as static HTML sites. From programming database queries and server deployments to creating graphics and writing CSS, I’ve touched just about every part of the application life cycle.

adminme.nwie.net

This internal Nationwide is called Adminme. It allows engineers to elevate their privileges to certain machines. Using Python Flask and BackboneJS we were able to create and deploy a robust tool able to communicate with multiple domains. This site was required to be IE10 and Firefox compatible.

Technologies Used: Python, Flask, BackBoneJS, SASS, PostgreSQL, Photoshop

This site was deployed to a Windows server using IIS and FastCGI with the website using a PostgreSQL DB.

Project Year: 2016

cobralabs.nwie.net

CobraLabs is an internal Nationwide site that displays basic information about the services we provided as well as a form to allow customers to request automation consultation. The frontend theme we designed and developed allowed us to quickly writeframe new projects and conformed to Nationwide brand standards.

Technologies Used: Python, Flask, BackBoneJS, SASS, SQLite, Photoshop, Illustrator

This site was deployed to a linux server using Apache with the website using a SQLite DB.

Project Year: 2015

Automation Work

During my time working within Infrastructure and Operations the teams I were apart of focused automating redundant tasks and finding ways to ways to give time back to people. RocketChat Urban Code Automation

Using RocketChat and HP Orchestration Operations I created a ChatBot and automation flow that allows the users to get information, create and modify Urban Code Teams.

Technologies Used: Python, Hubot, HP Orchestration Operations, NodeJS

Project Year: 2018

Selenium Browser Automation

The Nationwide Security Team required user information from the rendered Qualys website. A self contained tool was built using Python and Selenium saving them 8-16 hours of month.

Technologies Used: Python, Selenium, PyInstaller, PhantomJS

Project Year: 2016

Layout and Branding

Mobile App Graphics

I was contracted to convert the graphics from a 2005 Flash game to scalable HD graphics for mobile devices. I was able to expedite the scaling of the graphics by using Python. Each game level and scene had to be recreated in Photoshop and PhaserJS. The mobile app was deployed using the Ionic Framework. The Game: Barter Island

Technologies Used: Photoshop, Sketch, PhaserJS, Ionic, Python CarioSVG

Project Year: 2017
